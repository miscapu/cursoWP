Estrutura de Pastas no WP

Ele tem 3 pastas importantes chamadas:

wp-admin
wp-includes
wp-content

Existem 2 arquivos que são a base para um tema, então são os arquivos obrigatórios para que nosso tema funcione:
1. index.php: Ponto de entrada para nosso template,

2. style.css


Nota Importante

O arquivo Single podemos visualizar um post único

O arquivo Page podemos ver uma página Estática

O arquivo index é a porta de entrada para nosso WordPress
