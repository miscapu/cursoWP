For create theme is necessary create a new folder in path:
wp-content/themes/name-yout-theme

1. The template name must be in lowercase
2. Create style.css in this format:
/*
Theme Name: MiSCapu
Theme URI: https://miscapu.com/themes/miscapu/
Author: MiSCapu
Author URI: https://miscapu.com/
Description: Start Theme WordPress.
Requires at least: 5.9
Tested up to: 6.0
Requires PHP: 5.6
Version: 1.0
License: GNU General Public License v2 or later
License URI: http://www.gnu.org/licenses/gpl-2.0.html
Text Domain: miscapu
Tags: one-column, custom-colors, custom-menu, custom-logo, editor-style, featured-images, full-site-editing, block-patterns, rtl-language-support, sticky-post, threaded-comments
*/

****** Important *****
Text Domain is exactly the folder name of our template
The tags are important for SEO of our theme for find our theme more easily
